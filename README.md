# Mon Portfolio

Ce projet est mon portfolio personnel, construit avec Next.js et déployé sur GitHub Pages.

## Développement local

```bash
npm run dev
```

## Déploiement

Le déploiement se fait automatiquement via GitHub Actions quand du code est poussé sur la branche main.

## Technologies utilisées

- Next.js
- React
- Tailwind CSS
- Framer Motion
- EmailJS
